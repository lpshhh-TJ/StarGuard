## 简介

在老龄化趋势下，独居老人数量日益增长，家中跌倒问题愈加多发。为了让子女及时了解老人的紧急状况和生活习惯，同时最大程度保护老人隐私，**StarGuard 智护星居**团队打造了一款以室内定位为核心的全流程解决方案。

### 核心优势

| 特性 | 技术方案 |
|------|----------|
| **精准定位** | 星闪 (SLE) 测距，穿透能力强，抗干扰能力强 |
| **跌倒检测** | CNN 测距 + ResNet 姿态识别，双重融合判断 |
| **隐私保护** | 无摄像头，仅通过无线信号和传感器监测 |
| **智能分析** | 大模型分析生活习惯，提供健康建议 |

## 快速开始

### 前置要求

| 组件 | 要求 |
|------|------|
| 硬件 | BearPi-Pico H2821 开发板 x 7 |
| 电脑 | 香橙派/Windows 电脑 + 串口连接 |
| Python | 3.9+ + MindSpore |
| Node.js | 16+ |

### 部署步骤

```bash
# 1. 克隆项目
git clone https://github.com/username/project.git
cd mindspore_submit

# 2. 硬件端 - 烧录固件
cd firmware
# 按照 firmware/README.md 指导完成环境搭建和固件烧录

# 3. 本地服务端 - 安装依赖
cd ../local_serve
pip install -r requirements.txt
# 配置 launch.bat/launch.sh 中的虚拟环境和串口
# 运行 launch.bat/launch.sh

# 4. 网页服务端 - 安装依赖
cd ../web_serve
npm install
node serve.js
# 访问 http://localhost:3000
```

## 技术栈

### 硬件端

- **开发板**: BearPi-Pico H2821
- **通信**: 星闪 (SLE) 无线测距
- **传感器**: 六轴 IMU

### 本地服务端

- **深度学习框架**: MindSpore
- **测距模型**: CNN (多重卷积核 + 残差连接)
- **跌倒检测**: ResNet (一维残差时序姿态识别)
- **滤波算法**: RANSAC + 卡尔曼滤波

### 网页服务端

- **后端**: Node.js + Express
- **前端**: Three.js 3D 渲染
- **大模型**: 生活习惯分析与健康建议
