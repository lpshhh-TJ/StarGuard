# 项目名称

星闪护佑

## 制作者团队

StarGuard智护星居

## 简介

在目前国内的老龄化趋势下，独居老人数量日益增长，家中跌倒问题也愈加多发，为了让子女及时知道独居老人的紧急状况，以及更好了解独居老人的生活习惯，同时最大程度保护独居老人的隐私，Starguard智护星居团队打造了一款以室内定位为核心，贯通硬件端-本地服务端-网页服务端的全流程解决方案。

其中硬件端设备包含给独居老人使用的腰带以及室内部署的基站，采用星闪测距技术，发挥其穿透能力强和抗干扰能力强的优势，为精准定位提供坚实基础；同时采用六轴IMU，精确捕捉用户的移动状态，进一步提高跌倒检测的准确率。硬件使用BearPi-Pico H2821开发板，硬件部分代码见firmware文件夹。

本地服务端采用多种预处理算法，以及具有多重卷积核和残差连接的CNN网络来处理星闪采集的多频点正交电磁波数据，采用回归任务来预测获得距离，并且加入多种滤波算法与CNN配合来抵抗多径传播带来的相位偏差；对于IMU数据则采用LSTM模型，通过捕捉各项加速度和角加速度随时间的变化来准确判断垂直于地面方向上相对位置的剧烈变化，并且与通过定位获取的绝对位置变化相结合，实现对“跌倒”这个动作的精确识别。本地服务端代码见local_serve文件夹。

网页服务端直观地向用户展示腰带佩戴者在室内的位置，并且在用户跌倒时能够及时在网页上更新。除此以外，网页上还会显示腰带佩戴者每天在各个房间活动的时间，并接入大模型来分析独居老人的生活习惯，进而根据实际情况给出不同的健康建议。网页部分代码见web_serve文件夹。
