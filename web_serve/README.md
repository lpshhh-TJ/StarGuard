# 星闪护佑 - 网页可视化服务端

基于 Three.js 的 3D 可视化监护系统，提供实时位置监控、摔倒报警展示和数据集采集引导功能。

**配套服务**: [本地服务端文档](../local_serve/README.md)

---

## 快速开始

```bash
cd web_serve
npm install
node serve.js
```

访问 http://124.70.161.112:3000/

## 系统架构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              系统架构                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  硬件基站                    串口数据                                        │
│      ↓                                                                     │
│  本地服务端 (local_serve)  计算 + 跌倒检测                                  │
│      ↓                              HTTP API                               │
│  网页服务端 (web_serve)      ←──  位置/状态                                 │
│      ↓                                                                     │
│  浏览器 (Three.js)          3D 可视化展示                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 功能特性

| 功能 | 描述 |
|------|------|
| **3D 实时监控** | 可视化展示老人在房间内的实时位置 |
| **摔倒报警** | 实时显示摔倒状态，红色高亮报警 |
| **基站配置** | 自定义 6 个基站锚点坐标 |
| **房间设置** | 配置房屋尺寸和房间区域划分 |
| **数据集采集** | 5x5 网格自动生成采集坐标，网页引导采集 |
| **俯视图** | 2D 房间俯视图显示当前位置 |

## 技术栈

- **后端**: Node.js + Express + CORS
- **前端**: Three.js 3D 渲染 + Canvas 2D
- **通信**: REST API (轮询)

## API 接口

### 状态更新

```http
POST /update
Content-Type: application/json

{
  "x": 2.5,
  "y": 1.0,
  "z": 2.0,
  "isFallen": false
}
```

### 获取状态

```http
GET /status
```

### 基站管理

| 方法 | 端点 | 描述 |
|------|------|------|
| GET | `/anchors` | 获取基站坐标 |
| POST | `/anchors` | 更新基站坐标 |

### 房间配置

| 方法 | 端点 | 描述 |
|------|------|------|
| GET | `/roomSize` | 获取房间尺寸 |
| POST | `/roomSize` | 更新房间尺寸 |

### 数据集采集

| 方法 | 端点 | 描述 |
|------|------|------|
| GET | `/dataset-start?height=1.0` | 开始数据集采集 |
| GET | `/dataset-coordinate/start` | 开始采集当前坐标 |
| POST | `/dataset-coordinate/complete` | 完成当前坐标 |
| POST | `/dataset-coordinate/skip` | 跳过当前坐标 |
| POST | `/dataset-stop` | 停止采集 |
| GET | `/dataset-status` | 获取采集状态 |

## 使用说明

### 3D 视图操作

| 操作 | 功能 |
|------|------|
| 左键拖拽 | 旋转视角 |
| 右键拖拽 | 平移视角 |
| 滚轮 | 缩放 |

### 配置步骤

1. **基站配置** - 点击「设置基站锚点坐标」，修改 6 个基站的 (x, y, z)
2. **房间尺寸** - 设置整体空间的长、高、宽
3. **房间区域** - 配置客厅、卧室、厨房、卫生间的划分范围

### 数据集采集流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              数据采集流程                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  1. 设置采集高度（默认 1.0 米）                                             │
│      ↓                                                                     │
│  2. 点击「采集数据集」生成 5x5 网格坐标                                      │
│      ↓                                                                     │
│  3. 移动到当前坐标点（黄色高亮）                                            │
│      ↓                                                                     │
│  4. 点击「开始」采集当前点数据                                              │
│      ↓                                                                     │
│  5. 等待采集完成，自动跳转下一个坐标                                        │
│      ↓                                                                     │
│  6. 重复步骤 3-5，完成全部 25 个坐标点                                      │
│      ↓                                                                     │
│  7. 运行 gather_and_gen_label.bat 汇总数据集                               │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 与本地服务端集成

本地服务端 (`local_serve`) 通过 HTTP POST 请求将位置和状态数据发送到此服务：

```python
# local_serve 示例代码
import requests

def update_web_server(x, y, z, is_fallen):
    requests.post('http://localhost:3000/update', json={
        'x': x,
        'y': y,
        'z': z,
        'isFallen': is_fallen
    })
```
